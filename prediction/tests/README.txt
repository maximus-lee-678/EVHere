***GETTING STARTED ON OPTIMIZATION FOLDER***
1. install all dependencies from requirements.txt
2. Create an account in WandB and use wandb.login(). Input API key when prompted
3. Create a sweep and import search_range.yaml file
4. Download pickle file from google drive into folder
https://drive.google.com/file/d/1jK2gjHBrOWVzAitgC9GQ08wbFYyRNgC3/view?usp=sharing
4. Run sweep

***GETTING STARTED ON EVALUATION FOLDER***
1. install all dependencies from requirements.txt
2. run python command "python evaluate_models.py"
